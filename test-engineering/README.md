# test-engineering skill

Claude Code skill for test strategy, automation planning, and quality engineering.

## What it covers

- Test pyramid design (unit/integration/E2E distribution)
- Automation decision quadrant and ROI estimation
- Selection criteria matrix with weighted scoring
- Test naming conventions and patterns
- Quality metrics and coverage goals
- Anti-patterns and troubleshooting
- Actionable workflow with output template

## Install

Copy `test-engineering/` into `~/.claude/skills/`.

Or install via:

```bash
npx skills add Lu1sDV/skillsmd
```
